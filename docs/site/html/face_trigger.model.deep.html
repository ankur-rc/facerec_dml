embed(images=None, landmarks=None)[source]¶

Generates embeddings for the given images. THe images should be a result of the face detector phase, i.e these images should contain a face detected by the face detector.

Parameters:
  • images (list of numpy.nadarray: (num_images, image_height, image_width)) – the images to get embeddings of
  • landmarks (list, shape: (num_images, 5, 2)) – the facial landmarks of the images
Returns:

the face embeddings
Return type:

list

Note: The images contain the entire frame, and not just the cropped out face. Alignmnet is taken care of when we generate the embeddings.
